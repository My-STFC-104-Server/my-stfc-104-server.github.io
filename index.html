ROE # Rules of Engagement (Server 104)

  1. Survey ships that are actively mining may not be attacked while they are UPC
  2. If hitting survey ship on a 0 node, requires to Proof from attacker, this can be done by either an ingame message or a screenshot
  3. No PvP in Token Systems
  4. No PvP against players in armada circles of the same alliance

ROE 1-4 do not apply when officially engaged in war, territory takeovers/defences or PvP events at designated systems.

Code of Conduct: All players commit to treat each other respectfully. Toxic behaviour such as insults, personal threats, sexist or racist behaviour have no place in a game and will not be tolerated. The code of conduct ALWAYS applies.
